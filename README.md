# azure-subscription-design
Praxisnahe Referenzarchitektur für Azure Subscription Design. Die Gegenüberstellung zeigt, wann Workloads getrennt und wann Shared Subscriptions sinnvoll sind. Berücksichtigt Governance, RBAC, Netzwerksegmentierung, Betrieb und Sicherheit nach den Prinzipien des Microsoft Cloud Adoption Framework (CAF).
